Midterm Task 5: Using Inline and Internal CSS

Instructions/Requirements

  1. Open your resume Lab Task 1.
  2. Apply the ff: requirements
    1. Use Internal CSS and appropriate css selector to do the ff:
      1. Set the body background color to any color you want
      2. Adjust the body margins to 100px (top right left and bottom)
      3. Set the appropriate font-family for the text content of your resume
    2. Use the class selector for all Heading 2 tags used in your resume and apply any appropriate color that you want
    3. Style the image with a border and center on the page (use inline style)
    4. Below the image (Center the full name, address, email, and contact number height set to 12) – Use id selector
    5. You may freely add appropriate CSS for the paragraphs and list content of your resume
    6. Apply appropriate styles for the table as well. (use id selector)
    7. Link Requirements:
      1. Create an Email link to your email.
      2. Create a link within the page using href and id selector for the ff:
        • Summary
        • Education
        • Experience
        • Hardware Skills
        • Software Skills
      3. Remove the underline (text-decorations set to none) for all text links
      4. Set a different color for the ff: link state:
        • a:link - a normal, unvisited link
        • a:visited - a link the user has visited
        • a:hover - a link when the user mouses over it
        • a:active - a link the moment it is click
    8. At the bottom of the resume add a Back to Top Link (using href and id selector)
